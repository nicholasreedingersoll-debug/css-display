Inline vs. Block Elements

By default, HTML elements are either block elements or inline elements.

Block elements, like the p tag this text is in, start on a new line and take up the entire width of their container. You can see that using the browser inspection tool.

Inline elements, like this text in an italics tag, continue on the same line and only take up as much space as they need. Also, inline elements are not affect by top and bottom margin.

You can change whether an element is block or inline using the CSS display property. Often, you might want to change inline elements to block elements to help with positioning and layout. As a general rule inline elements should NOT contain block elements, but block elements can contain inline or block elements.
